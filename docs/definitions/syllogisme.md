# Syllogisme
Syllogisme: On part de
- une majeur - "Tout les hommes sont mortels"
- une mineure - "Socrate est un homme"
- une conclusion - "Donc, Socrate est mortel"

Syllogisme:
- Tout ce qui est rare est cher.
- Un éléphant rose est rare.
- Donc, un éléphant rose est cher.

NPC: la forme $\neq$ Le fond
