# Concept
Idée générale abstraite

Universe -> propre la la totalité d'une classe d'elements, sans exception
Générale -> propre à une série d'élement mais admet des exceptions.
Particulier -> propre à un ensemble d'élement, dans une classe d'objets.
Singulier -> renvoie à l'unité, propre à un élement
